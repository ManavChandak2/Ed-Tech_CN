# Coding Ninjas Web Project

This project is a **multi-page educational website** created using **HTML only**, as part of the Coding Ninjas Frontend Fundamentals module.
It demonstrates the basic structure of a website with navigation, embedded media, forms, and data tables.

---

---

## üåê Pages Overview

### 1. **Home Page (`index.html`)**

- Displays Coding Ninjas introduction, about section, and featured courses.
- Each course image links to the respective video or course section.
- Includes testimonials and social media icons (linked to Coding Ninjas official pages).
- The Coding Ninjas logo redirects to the Home Page from all other pages.

### 2. **Courses Page (`courses.html`)**

- Displays four main courses:
  - Web Development
  - Data Science Fundamentals
  - Digital Marketing Mastery
  - Android Development
- Each course section includes a short description, embedded video, and **"Enroll Now"** button linking to the Register Page.

### 3. **Register Page (`register.html`)**

- Contains a form to collect user details:
  - Full Name, Email, Username, Password
  - User Type (Student, Instructor, Admin)
  - Course Selection dropdown
- Submit button labeled **"Register"**.

### 4. **Login Page (`login.html`)**

- Simple login form with:
  - Username and Password fields
  - User Type dropdown
  - **Login** button and link to register for new users.

### 5. **Contact Us Page (`contact.html`)**

- Displays contact details:
  - Email and phone (clickable links)
- Embedded Google Map for Coding Ninjas‚Äô Gurgaon office.
- Feedback form to collect user queries (Name, Email, Subject, Message).

### 6. **Admin Dashboard (`admin.html`)**

- Displays user enrollment data in a structured table:
  - User ID, Full Name, Email, Enrolled Courses (Course ID + Title)
- Table footer shows total enrollments.
